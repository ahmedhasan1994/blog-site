Html

Hypertext Markup Language is a markup language used to create and design web pages and websites. This language is considered one of the oldest and most widely used languages ​​in web page design.

Css

Style Sheets (abbreviation: CSS) It is a formatting language for web pages that is concerned with the form and design of websites. It was specifically designed to separate formatting from the content of the written document. This applies to the colors, fonts, images, and backgrounds that are used on the pages, with complete flexibility and ease. This technology is concerned with the overall appearance of website pages, including colors, images, etc.

JavaScript

JavaScript, commonly known as "JS", is a high-level programming language, primarily used in web browsers to create more interactive pages. It is currently being developed by the TC39 committee, which is managed by the ECMA standards organization. JavaScript began to be used as a programming language aimed at amateur and non-professional programmers.
